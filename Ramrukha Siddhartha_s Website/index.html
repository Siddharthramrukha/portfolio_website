My Reflection

This reflection is about the challenges I have faced when creating my website
and how I overcame them.

1. Designing the website. Designing a website that is have a nice design and user-friendly. It's important to keep the design simple, using appropriate colors and text that matches with the theme.

2. Difficulties to write the right design and text size in CSS. I have done a lot of research on the W3Schools HTML Styles - CSS website to learn about how to fix it and to manipulate the layout of my webpage.

3. Illegal characters. Illegal characters in path segment example "Week 1.png". W3School HTML validator have detected this error which means that there must not be any space in the files name. Many other errors were detected by the Markup Validation Service which have been fixed.

4. Making sure it is responsive. It was the most challenging part as I am not used to make responsive website. But this has not stopped me from learning new things so, I went to W3Schools HTML Responsive Web Design website to learn about media queries and keyframes. It's important to use responsive design that adjusts to different screen sizes so that users can access it to different devices.

5. I have learned how to make and style a download button so that visitors can get access to my resume easily.
